## Project Description
The Weather Dashboard is a simple web application that allows users to search for a city and view the current weather information.  
It displays the city name, temperature, and weather condition using data fetched from a public weather API.

This project was created to practice Git collaboration, GitHub Actions CI/CD pipelines, and cloud deployment.
